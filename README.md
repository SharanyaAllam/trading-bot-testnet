## Overview
This project is a CLI-based Python trading bot that places Market and Limit orders on Binance Testnet.

## Features
- BUY and SELL orders
- Market and Limit order support
- Command-line interface (CLI)
- Input validation
- Logging of API requests and responses
- Error handling for invalid input and API failures

## Tech Stack
- Python 3.x
- python-binance
- argparse
- logging
